# Sales-Performance-Dashboard-
Overview
An interactive Power BI dashboard built for sales performance analysis, spanning three dashboard pages covering key business metrics, trends, and breakdowns to support data-driven decision-making.
Tools Used
Power BI (Data Modeling, DAX, Report Design)
DAX (Data Analysis Expressions)

What the Dashboard Covers
Page 1 – [Business overview]: [ high-level KPIs — total sales, revenue, growth trends]
Page 2 – [Manager/Category performance]: [time-based sales trends, YoY/MoM comparisons]
Page 3 – [Returns and Delivery Delays]: [ product/region/category-level performance]
